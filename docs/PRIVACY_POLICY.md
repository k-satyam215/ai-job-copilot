# Privacy Policy Draft

AI Job Copilot processes resume, profile, job, and application data to provide job matching, form autofill, resume optimization, and career analytics.

Data processed:

- Account information: name, email, phone.
- Resume content and parsed profile.
- Job pages visited while using the extension.
- Application status and generated answers.
- Copilot memory such as salary preference, notice period, and location preference.

The browser extension communicates with the user's configured backend API. Data is not sold. Production deployment should add retention controls, deletion export, audit logs, and provider-specific DPAs.

Users should review all AI-generated content before submitting applications.
